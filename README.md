
## Requirements


- PHP version: 8.2
- Composer version: 2.6.5
- NPM version: 10.1.0
- Node version: 20.8.1

## Database Information

- **DB:backend_panel**
- **DB_USER:root**

## Laravel Installation

- **composer install**
- **cp .env.example .env**
- **php artisan key:generate**
- **php artisan migrate**
- **php artisan serve**

## React Installation
- **npm install**
- **npm run dev**
